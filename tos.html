4. Commands & Features

VoKi includes a variety of commands:

  • Moderation: ban, unban, kick, mute, unmute, add role, remove role, set logs channel, set support role
  • Utility: relay, support, say
  • AI Tools: ask (AI-powered responses)
  • Server Management: rank, set welcomer
Commands may update over time.

5. Event Listeners

VoKi uses Discord event listeners for certain features:

  • messageCreate: leveling (rank) & AI ping (ask)
  • guildMemberAdd: set welcomer
  • guildMemberRemove: set logs channel
  • messageUpdate: messages edited (in development)

6. Data We Collect

  • Discord User ID & Server ID for command execution and permissions.
  • Message content only when needed for relay or AI commands.
  • Server configurations: roles, channels, and settings.
  • Interaction logs for debugging and ensuring service reliability.

7. How We Use Your Data

Data is used solely to provide and improve VoKi. We do not sell or share your personal data. VoKi admins cannot view your data; they may only delete it upon request. Users may opt out to remove all saved data.
