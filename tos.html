Privacy Policy

1. Privacy-First Principle

Nora is designed with privacy as a core principle. Server data belongs to the server and its members, not Nora, its developers, or its founder.

2. Data Collected

Nora collects only minimal data required to function. This may include server IDs, channel IDs, role IDs, command configuration, and feature-specific data.

3. Leveling System Data

If enabled, Nora processes limited activity indicators (such as message events or participation triggers) to calculate user levels. This data exists only within the server where it was generated and is not shared or aggregated globally.

4. Welcome & Join Events

When the welcomer feature is enabled, Nora temporarily processes Discord join events to send welcome messages. This may include mentioning the joining user and displaying their Discord avatar. No join data is sold, profiled, or reused.

5. Message Content

Message content may be processed in-memory for moderation or AI context but is not permanently stored unless explicitly required for a configured moderation feature.

6. No Developer Access

Nora developers and its founder do not view, monitor, or access server messages or private user data.

7. Data Isolation

Each server’s data is isolated. Data from one server is never visible to or shared with another server.

8. Data Sharing

No data is sold, rented, or shared with third parties. Nora does not use server data for advertising or analytics beyond service operation.

9. Data Deletion

Server owners may delete configuration or feature data through commands or by removing Nora from the server. Removal of the bot deletes associated server data.

10. Policy Updates

This Privacy Policy may be updated to reflect feature changes or legal requirements. Updates will be communicated through official Nora channels.
