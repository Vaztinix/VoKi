Terms of Service & Privacy Policy

Welcome to VoKi. By using our services, you agree to the following terms and policies.

1. Purpose

  • Provide privacy-first AI moderation tools for Discord communities.
  • Enhance server safety, trust, and fairness through automation.
  • Support secure, positive interactions among all users.

2. Responsible Use

  • Use VoKi only for intended purposes and respect others.
  • No harassment, spamming, or unauthorized data collection.
  • Compliance with Discord Terms of Service is required.
  • Admins configure bot according to server rules.

3. AI Features

  • AI assists moderation and support; human oversight is required.
  • Short-term context memory improves accuracy.
  • Moderators can review, adjust, or override AI actions.
  • Transparency and fairness are prioritized.

4. Privacy Policy

  • No personal data collected without consent.
  • All data remains on your server; developers do not access it.
  • Users may view and delete data with the User Data command.
  • Removed logging of message edits, deletions, and leaves to protect privacy.

5. Permissions

  • VoKi requests only necessary permissions for moderation.
  • Server admins control bot access and privileges.
  • Command management per server coming soon.

6. Guidelines

  • Follow community standards and respect VoKi’s systems.
  • Report bugs, misuse, or requests responsibly.
  • Admins should review permissions periodically.

7. Logging

  • Moderation actions logged locally for transparency and accountability.
  • No sensitive message content is stored externally.
  • Logs help in dispute resolution and management.

8. Security

  • Secure coding practices and permission handling applied.
  • Admins should review access and commands periodically.
  • Server interactions minimize data exposure.

9. Updates

  • VoKi receives updates to improve functionality, AI moderation, and security.
  • Major changes communicated via Discord announcements.
  • Roblox integrations removed; focus is on Discord moderation.
  • Multilingual support temporarily paused for improvements.

10. Support

Visit our support server for assistance.
